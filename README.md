# AutoFrame 图片排版

AutoFrame 是一个本地运行的图片自动排版网页工具。用户可以在画布中调整矩形区域，把多张图片加入区域后，工具会自动计算图片布局，并支持导出 PNG。

## 功能

- 支持本地添加多张图片，图片不上传服务器
- 支持拖拽调整排版区域大小和位置
- 支持区域比例预设：自由、1:1、4:3、16:9、自定义
- 支持三种布局模式：拼贴、网格、行列
- 支持图片适配方式：裁切填满、完整显示
- 支持设置图片间距、圆角和区域背景色
- 支持控制完整显示模式下的空白上限和填满方向
- 支持随机板式、反转顺序、清空图片
- 支持导出 PNG，并可选择标准、高清、超清导出倍率

## 界面设计

- 工具栏采用分区式控制面板：工具、画布区域、排版和导出分开管理。
- 品牌强调色统一为 Orange Coral 渐变，用于 Logo、选中按钮、画布控制线和关键高亮状态。
- 普通操作按钮保持浅色样式，避免和选中态抢视觉焦点。
- 默认区域背景为灰白色，默认间距和圆角均为 `0`，网格背景默认开启。
- 区域比例只展示常用预设，自定义比例和比例锁定收进高级控制。
- 页面底部保留开发者署名和用途简介，说明保持简洁。

## 使用方式

直接用浏览器打开 `index.html`。

也可以在项目目录启动一个本地静态服务：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 核心排版规则

AutoFrame 当前支持两类图片显示策略：

- 裁切：每张图片填满自己的排版格，允许裁切边缘，视觉上更容易铺满整个区域。
- 完整：每张图片保持原始比例并完整展示，优先减少分散空白；无法完全铺满时，通过行或列方向把剩余空白集中处理。

完整显示模式的目标规则是：

1. 保持每张图片原始宽高比。
2. 不拉伸图片。
3. 尽量避免图片之间出现零散空白。
4. 如果无法完全填满矩形区域，剩余空白应尽量集中在顶部、底部、左侧或右侧。

## 技术说明

该项目是纯前端静态网页，不依赖构建工具。主要技术包括：

- HTML Canvas 绘制画布和图片
- File API 读取本地图片
- Canvas 导出 PNG
- 原生 JavaScript 管理布局状态和交互

## 隐私

图片文件只在浏览器本地读取和绘制，不会上传到远程服务器。
